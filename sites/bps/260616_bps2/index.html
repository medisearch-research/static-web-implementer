1차(bps) → 2차(bps2) — 1차는 "BEAUTY BACKED BY PHARMACY" 키네틱 타이포가 전면을 덮는 bp-plus Manus 랜딩 스타일이라 hince 샵 구성과 무관했습니다. 2차는 hince 홈 골격을 따릅니다:
· 헤더: 햄버거(좌) · 중앙 BP+ 로고 · 검색/계정/장바구니 라인 아이콘(우) + 공지바(X 닫기)
· 히어로: Swiper 제품 배너(라이트 톤·슬라이드 카운터·닷) — 키네틱은 브랜드 슬라이드 1장만 절제
· 원형 카테고리 → BEST SELLER 4열 랭킹(핑크 랭크배지·할인%·별점·색상칩·상태뱃지) → 비대칭 NEW ARRIVALS
· 피부고민 탭 → 라이트 핀 브랜드 스토리 → 약사의 노트 → 프로모 → 밝은 시네마틱 → 리뷰 → signature 배경 + 초대형 BP+ 워드마크 푸터

※ 폰트·GSAP·Swiper는 CDN 로드(인터넷 필요). 제품 비주얼은 CSS 목업(라이선스 실사 스왑 지점 주석). 상품/수치는 데모 예시.
